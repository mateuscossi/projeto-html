HTML

Oque é o HTML

O HyperText Markup Language conhecido como HTML é uma linguagem de marcação utilizada para criar páginas na web. Ele estrutura o conteúdo de uma página, definindo elementos como cabeçalhos, parágrafos, links, imagens e outros componentes. O HTML usa uma série de tags para indicar esses elementos, permitindo que navegadores da web saibam como exibir o conteúdo corretamente.

A primeira versão do HTML era bastante simples, com poucas tags e funcionalidades básicas. No entanto, à medida que a internet cresceu e evoluiu, o HTML também se desenvolveu, ganhando novas tags e capacidades. Hoje, o HTML é uma linguagem de marcação robusta e essencial para a criação de páginas web, sendo continuamente atualizado pelo World Wide Web Consortium (W3C).

Criação do HTML

O HTML foi desenvolvido por Tim Berners-Lee no final dos anos 1980 e início dos anos 1990. Tim Berners-Lee é um físico e cientista da computação britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear) na época. Ele queria encontrar uma maneira de facilitar a troca e a estruturação de informações entre os pesquisadores.

Em 1990, Berners-Lee escreveu a primeira versão do HTML, juntamente com o primeiro navegador web chamado WorldWideWeb, e o primeiro servidor web. Ele se baseou em conceitos de outros sistemas de marcação de texto, como SGML (Standard Generalized Markup Language) e o projeto ENQUIRE.

HTML e CSS

HTML é frequentemente usado em conjunto com CSS (Cascading Style Sheets) para definir o estilo visual das páginas web. Enquanto o HTML estrutura o conteúdo, o CSS controla a aparência, como cores, fontes e layout. É basicamente o CSS que faz o seu site ficar com uma cara bonita.

HTML e JavaScript

JavaScript é outra linguagem muito usada com HTML também, porém ele funciona para adicionar interatividade às páginas web. Com JavaScript, é possível criar efeitos dinâmicos e responder a ações do usuário, como cliques, movimentos do mouse, menus interativos e muitas outras coisas.

Elementos Principais do HTML

  • Cabeçalhos: <h1> a <h6>
  • Parágrafos: <p>
  • Links: < a href=" URL " >
  • Imagens: <img src="imagem.jpg" alt="Descrição da imagem">


Versões do HTML

HTML 1.0

  • Lançado: 1993
  • Características: A primeira versão do HTML era bem básica, com suporte para links, parágrafos e cabeçalhos. Não tinha muitos elementos de formatação ou estilo, mas foi um grande passo para a criação de páginas web.

HTML 2.0

  • Lançado: 1995
  • Características: Esta versão trouxe melhorias significativas, como suporte para formulários, tabelas e a capacidade de inserir imagens. O HTML 2.0 padronizou muitos dos elementos introduzidos no HTML 1.0.

HTML 3.2

  • Lançado: 1997
  • Características: O HTML 3.2 adicionou suporte para scripts (JavaScript), applets e estilização básica com o uso de tabelas. Foi a primeira versão a ser amplamente adotada e utilizada na criação de páginas web interativas.

HTML 4.01

  • Lançado: 1999
  • Características: Esta versão foi um grande avanço em termos de estrutura e acessibilidade. Introduziu o conceito de separação entre conteúdo e apresentação, incentivando o uso de folhas de estilo em cascata (CSS). O HTML 4.01 tinha três variantes: Strict, Transitional e Frameset.

HTML5

  • Lançado: 2014
  • Características: O HTML5 é a versão mais moderna e poderosa do HTML. Ela trouxe suporte nativo para vídeos, áudios e gráficos, sem a necessidade de plugins externos. Além disso, introduziu novas tags semânticas, como <header>, <footer>, <article> e <section>> melhorando a estrutura e a acessibilidade das páginas web. O HTML5 também melhorou o suporte para dispositivos móveis e integração com APIs modernas.
